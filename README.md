# CajuScript

Fast scripting language for the JVM, compiles to binary class, and full customizable.

## Documentation

- [Here is the documentation.](https://github.com/cajuscript/cajuscript/blob/main/doc/README.md)

## Compilation

```
mvn compile
```
### Build

```
mvn package
```
