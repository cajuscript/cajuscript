# Arithmetics Operators

The arithmetics operations supported and their operators.

## Addition

```
x = 0 + 1
x += 1
```

> `++` is not supported. Use `+=1` instead.

## Subtraction

```
x = 0 - 1
x -= 1
```

> `--` is not supported. Use `-=1` instead.

## Multiplication
```
x = 0 * 1
x *= 1
```

## Division
```
x = 0 / 1
x /= 1
```

## Modulus
```
x = 0 % 1
x %= 1
```
