# Cast

Casting objects

## Bases

```
newObject = caju.cast(OBJECT, "TYPE")
```

## Types

> The string with type name is case-insensitive.

Integer type:

```
caju.cast(obj, "int")
caju.cast(obj, "i")
```

Long type:

```
caju.cast(obj, "long")
caju.cast(obj, "l")
```

Float type:

```
caju.cast(obj, "float")
caju.cast(obj, "f")
```

Double type:

```
caju.cast(obj, "double")
caju.cast(obj, "d")
```

Char type:

```
caju.cast(obj, "char")
caju.cast(obj, "c")
```

Boolean type:

```
caju.cast(obj, "boolean")
caju.cast(obj, "bool")
caju.cast(obj, "b")
```

Byte type:

```
caju.cast(obj, "byte")
caju.cast(obj, "bt")
```

String type:

```
caju.cast(obj, "string")
caju.cast(obj, "s")
```

Any class type:

```
caju.cast(obj, "java.lang.Object")
```
